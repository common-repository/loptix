== Description ==

Loptix displays last post from loptix network. 

Loptix is free links sharing/traffic network.

== Installation ==

1. IMPORTANT! If you are upgrading from a previous version first deactivate the plugin, then delete the plugin folder from your server.
2. Upload the plugin folder to your /wp-content/plugins/ folder. 
3. Go to the **Plugins** page and activate the plugin.
4. Go to Appearance -> Widgets and Drag&Drop Loptix to Your Sidebar
5. Use the **Options/Settings** page to adjust the behaviour of the plugin.

[Loptix site](http://loptix.com/) has [full instructions](http://loptix.com/widget.php).
